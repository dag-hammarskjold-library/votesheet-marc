# votesheet-marc
Convert vote result PDF to MARC.

#### Usage:
> `perl run.pl`

#### Requirements:
Install [Strawberry Perl](http://strawberryperl.com/) using the `.msi` installer from the website.

Install `Win32::GUI`, `PDF::API2`, `Tie::IxHash`
> `cpanm install Win32::GUI PDF::API2 Tie::IxHash`
